# Online Event Registration System

This project is a simple web-based system that allows users to register for events online.  
It is created as part of the Software Project Planning and Management assignment using Jira, Confluence, and GitHub.

## Project Overview
The system contains three basic pages that represent the structure of an Online Event Registration System:
- Home page (index.html)
- Login page (login.html)
- Event registration form (register.html)

These pages are used to demonstrate planning, version control, and task tracking.

## Technologies Used
- HTML  
- CSS (optional)  
- JavaScript (optional)  
- GitHub for version control  
- Jira for task and issue tracking  
- Confluence for project documentation  

## Project Files
- **index.html** – Home page with links to login and registration pages.  
- **login.html** – Basic login form for users.  
- **register.html** – Event registration form for selecting and submitting event details.  
- **README.md** – Description and details about the project.  

## Project Management Approach
This project is managed using the following tools:
- Jira for creating tasks, tracking progress, and managing issues.
- Confluence for documenting the project plan, requirements, effort estimation, and reports.
- GitHub for maintaining source code and commit history.

## Basic Features
- Display available events.
- Allow users to register for an event.
- Simple user interface demonstrating core functionality.
